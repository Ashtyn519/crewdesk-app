# Assets

Place the following files here before building:

## images/
- icon.png (1024x1024) - App icon, amber background with lightning bolt
- splash-icon.png (1024x1024) - Splash screen icon
- adaptive-icon.png (1024x1024) - Android adaptive icon foreground
- favicon.png (32x32) - Web favicon

## fonts/
- SpaceMono-Regular.ttf - Available from Google Fonts

All images should use the CrewDesk brand colors:
- Background: #04080F
- Primary: #F59E0B (amber)
- Logo: Lightning bolt (Zap icon)
